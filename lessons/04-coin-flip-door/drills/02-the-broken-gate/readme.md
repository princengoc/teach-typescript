# 02: the broken gate

Three questions, three mistakes, one each. This drill is about reading the
error, not guessing. The editor underlines all three before you run anything;
hover the red line and read what it says.

1. Fix `isWide` so it says whether the gap is bigger than 0.
2. Fix `isBig` so it answers for every gap, not just the big ones.
3. Fix `isTwo` so it says whether the gap is the number 2.

Change as little as you can. Two of these are one character.
