# Hints

**Task 1.** "Type 'number' is not assignable to type 'boolean'." The function
promised a boolean and handed back a number. `isOpen` above it asks the same
question the right way. Read it.

**Task 2.** "Function lacks ending return statement." An `if` on its own
covers one of the two answers and leaves the other missing. The card has the
word for the block that covers it.

**Task 3.** "This comparison appears to be unintentional." `'2'` with quotes
round it is a word. `2` without them is a number. They are never the same
value, so the question can never be true.
