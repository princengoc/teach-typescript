# 01: true or false

Three questions, and one of them asks another. Work out what each one hands
back before you run anything.

1. Write what `isLong(4)` hands back into `answer1`.
2. Write what `isExactly(3, 3)` hands back into `answer2`.
3. Write what `isShort(9)` hands back into `answer3`.

Write `true` or `false`, not calls. `export const answer1 = isLong(4);` would
pass the test and teach you nothing.

Only task 1 runs at first. Delete the `.skip` on the next test when it passes.
