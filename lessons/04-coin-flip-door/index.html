Lesson 4: the coin-flip door

One shelf to paint, and a door that moves. Half the rooms have the door in the left wall, half in the right. The robot always walks in the same way: facing down the page, standing on the door.

Here is a straight-line move. It turns right, then paints the three squares in front of it. Run it in both rooms. 

robot.turnRight();
robot.paint();
robot.walk(1);
robot.paint();
robot.walk(1);
robot.paint();

The same move, and only one room survives it. Turning right is right half the time. Writing more calls will not help: the code has to look first.

So the robot gets one question. robot.wallOnLeft(); hands back a value that is only ever true or false. That kind of value is a boolean. Ask it in each room.

Two rooms, two answers, and now the code can tell them apart. if runs the block after it only when the answer is true. else names the block to run instead. 

if (robot.wallOnLeft()) {
  robot.turnRight();
} else {
  robot.turnLeft();
}
