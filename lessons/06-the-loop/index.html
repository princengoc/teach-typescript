Lesson 6: the loop

Last time you built a staircase by writing the same block three times. Here are two ways to say a block once and run it many times.

Way 1: a for loop

The three parts inside the brackets are start, keep-going test, step.

for (let i = 0; i < n; i += 1) {
  // the block runs, once each time round
}

i starts at 0. While i < n is true the block runs, then i += 1 steps it up. Watch it climb.

Way 2: recursion

A function can call itself. It does one step, then hands the rest to another copy of itself, with new numbers. A base case stops it. 

function countDown(n) {
  if (n === 0) return;   // base case: stop
  // one step here
  countDown(n - 1);      // the rest, one smaller
}

Nothing changes value. Each call gets a fresh, smaller number.
