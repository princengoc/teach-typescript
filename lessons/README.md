# Lessons

1. **fix the start** -- read values off an object to build the room.
2. **paint the L** -- make things happen by calling the robot, in order.
3. **name the move** -- package a list of calls under one name and reuse it.
4. **the coin-flip door** -- pick which lines run with `if` and `else`.
5. **the staircase** -- paint bars, each one taller than the last. The height is
   a value that grows: `let`, and `height += 1` after each bar. `const` cannot.
6. **the loop** -- say a block once and run it many times, two ways: a `for`
   loop and recursion. A ladder of robot figures -- square, rectangle, staircase
   -- each read from the room, so copy-paste cannot win.

Built: 1-6.
