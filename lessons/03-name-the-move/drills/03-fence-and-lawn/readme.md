# 03: the fence and the lawn

A square garden. One number describes it: `side`, how many metres long each
edge is.

1. `fenceLength(side)` hands back the metres of fence needed to go all the way
   round.
2. `lawnArea(side)` hands back the square metres of grass inside.

Both take the same argument and hand back a very different number. Say out
loud what each one does before you write it.
