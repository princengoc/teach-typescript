# 05: the room, again

Back to lesson 1 for a moment. A room is an object: two numbers that belong
together, each under its own name, read back out with a dot. This drill uses
no word from card 03 except the ones you just learned.

1. `floorArea(room)` hands back how many squares the floor has.
2. `paintCost(room)` hands back the cost of painting that floor, at 2 a
   square.

Write task 2 without repeating task 1's sum.
