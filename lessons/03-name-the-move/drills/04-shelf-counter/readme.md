# 04: the shelf counter

Part 1 has one test with one number in it. Part 2 has many. Write part 1 so it
would survive part 2 and you only do the work once.

1. `squares(shelves)` hands back how many squares get painted, when every
   shelf is 3 long.
2. `squaresOf(shelves, length)` hands back the same count for shelves of any
   length.

`squaresOf` takes two arguments. The order matters: shelves first, length
second, the same order as the names in the brackets.
