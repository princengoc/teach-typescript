# Hints

**Task 1.** `twice` does not do the work itself; it calls `add`. Read `add`
first, then put `4` in for `n` and see what `add` gets handed.

**Task 2.** `bigger` calls `twice`, which calls `add`. Start at the inside.

**Task 3.** Work out each argument on its own before you add them. The card
line for this is the one about `return`: the call stands for the value it
hands back.
