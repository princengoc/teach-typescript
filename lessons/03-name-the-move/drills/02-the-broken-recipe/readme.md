# 02: the broken recipe

Three functions, three mistakes, one each. This drill is about reading the
error, not guessing. The editor underlines all three before you run anything;
hover the red line and read what it says.

1. Fix `twoShelves` so it hands back the number of squares in two shelves.
2. Fix `oneShelf` so it counts a shelf 3 long.
3. Fix `shelfName` so it hands back `a shelf of 4` when you call it with `4`.

Change as little as you can. Two of these are one word.
