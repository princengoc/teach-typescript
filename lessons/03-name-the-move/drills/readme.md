# Drills

Five jobs, then two spare ones if you want more. No robot, no picture: just
functions that take something and hand something back.

Work in order. Each folder has a `readme.md` saying what to do, a file to
edit, a test file, and a `hints.md` for when you are stuck. The hints point at
the card. They never hand you the answer.

Only the first task in a test file runs. When it passes, delete the `.skip`
from the next one and keep going.

Run one drill from the terminal:

```
npx vitest run drills/01-what-it-says
```
