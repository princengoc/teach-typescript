Lesson 3: name the move

A shelf is three squares in a row. Paint one with the buttons below, and count your presses. The robot walks with robot.walk(1); — the 1 is how far.

One shelf is a lot of presses. Three shelves is three times that, and you would be writing the same calls out again and again.

So give the calls one name. That is a function: 

function paintShelf(): void {
  robot.paint();
  robot.walk(1);
  robot.paint();
  robot.walk(1);
  robot.paint();
  robot.turnLeft();
  robot.turnLeft();
  robot.walk(2);
  robot.turnLeft();
  robot.turnLeft();
}

Now the whole move is one press, and one line. The last four calls walk the robot back to where it started, so the next shelf can begin the same way.

You wrote the move once and spent it three times. That is the whole lesson. Now do it in the file.
