# Lesson 01: fix the start

**Start here:** look at the preview panel on the right. It shows the game and
tells you the one thing to change. Edit `src/exercise.ts` (open on the left),
save, and the preview updates. When its verdict turns green and reads `PASS`,
you have finished the main task.

You do not need the terminal for this lesson. The preview is your test.

## After the robot turns green

1. Read the front of `card.md` (the words behind what you just did).
2. Do the drills in order, `drills/01` to `drills/04`. Each folder has a
   `readme.md`; edit the one file it names.
3. Read the back of `card.md`, then tell your teacher what the robot did wrong
   before your fix, and why.

## Rules

- Edit only the files a readme names. `src/harness/` runs the world; leave it
  closed.
