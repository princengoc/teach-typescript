The Painter Robot

You program a robot that lives on a grid and paints squares. In this first lesson you build the room: you set its size, place a door, and put the robot where it should start. You do it all by changing numbers in src/exercise.ts.

Demo: what the robot does

Once the room is built, the robot follows a set of moves and paints as it goes. Press Play to watch it start on the door and paint the target. Then see what happens when it starts in the wrong place.

Lesson 1: build the room

Edit src/exercise.ts (open on the left), save, and watch here. Do it one value at a time:

  1. Size the room: change grid.width and grid.height. Watch it resize.
  2. Place the door: set door onto the dashed goal square.
  3. Start the robot: set startX and startY onto the door.

When the line above turns green and reads PASS, you are done.
