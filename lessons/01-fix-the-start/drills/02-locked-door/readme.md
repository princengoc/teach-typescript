# Locked door

`const` locks a name to its value. This drill makes you pick the lock the wrong
way first, so you meet the error that guards it.

1. Uncomment the marked line in `door.ts`. The editor underlines it. Read the
   error message: it names the constant and says why the line is refused.
2. Delete the line. Change the value where `doorCode` is born instead. Run
   `npm test`.

The error you met is on the lesson card, in the "this does not compile" column.
